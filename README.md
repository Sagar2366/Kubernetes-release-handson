# Kubernetes Release Hands-On

Hands-on demos, manifests, and scripts for Kubernetes releases. Each release gets its own folder with runnable examples you can use to learn, teach, or record videos.

## Releases

| Version | Codename | Release Date | Folder |
|---------|----------|-------------|--------|
| v1.36 | Haru | April 22, 2026 | [v1.36/](v1.36/) |

## How to Use

1. Pick a release folder (e.g. `v1.36/`)
2. Follow the README inside that folder for prerequisites and quick start
3. Each release has a `00-kind-cluster/` folder to spin up a local cluster
4. Run individual demo scripts in any order (see the release README for recommendations)
